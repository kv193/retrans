For run application^

1. Build project.
2. Connect Nucleo-F411RE to USB.
3. Run in command prompt "burn".

LED1 is additional led connected to ARDUINO CN9.D12 - CN9.GND
For properly work ST-Link necessary to write stlinkv*.rules to /etc/udev/rules.d
This give permission for access to usb device.
